## บริการหลัก (Service Keywords)
## เครื่องซักผ้าหยอดเหรียญ ขอนแก่น
- ให้คำปรึกษาเลือกเครื่อง + ติดตั้งระบบหยอดเหรียญ
- เดินระบบไฟฟ้า/ตู้เมน/ตู้ MDB (ตามความเหมาะสมของหน้างาน)
- เหมาะกับหอพัก อพาร์ตเมนต์ ชุมชน ธุรกิจบริการ  
👉 หน้าแนะนำ: `laundry.html`

## ตู้น้ำหยอดเหรียญ / ตู้น้ำ RO ขอนแก่น
- ติดตั้งตู้น้ำ RO + ระบบกรองน้ำ
- เดินท่อ/ไฟให้ปลอดภัย พร้อมใช้งาน  
👉 หน้าแนะนำ: `water.html`

## ระบบหยอดเหรียญ / ตู้แลกเหรียญ ขอนแก่น
- ติดตั้งระบบหยอดเหรียญสำหรับงานบริการ
- ประเมินหน้างานเพื่อให้ใช้งานจริง  
👉 หน้าแนะนำ: `vending.html`

## อินเทอร์เน็ต / ฮอตสปอตหอพัก ขอนแก่น
- ตั้งค่าหน้าล็อกอิน (Hotspot Login) แยกยูสเซอร์
- จำกัดความเร็ว/แพ็กเกจ และวางสัญญาณให้ครอบคลุม  
👉 หน้าแนะนำ: `internet.html`

## CCTV / คีย์การ์ด / ไม้กั้นรถ ขอนแก่น
- ติดตั้งกล้องวงจรปิด CCTV ดูผ่านมือถือ
- คีย์การ์ด ประตูแม่เหล็ก ลงเวลา ไม้กั้นรถ ประตูอัตโนมัติ  
👉 หน้าแนะนำ: `security.html`

## งานไฟฟ้า ขอนแก่น (แรงต่ำ-แรงสูง) / ตู้ MDB / หม้อแปลง
- เดินสายไฟ ตู้เมน ตู้ MDB งานปรับปรุงระบบไฟฟ้า
- รองรับงานระบบหยอดเหรียญและอาคาร  
👉 หน้าแนะนำ: `electrical.html`

## โซล่าเซล / แอร์ ขอนแก่น
- ประเมินหน้างานเพื่อประหยัดพลังงานจริง
- ติดตั้ง/ปรับระบบแอร์ให้เหมาะกับการใช้งาน  
👉 หน้าแนะนำ: `solar-air.html`

## รับสร้างบ้าน / ต่อเติม ขอนแก่น
- ต่อเติม ปรับปรุง ซ่อมแซม งานก่อสร้าง
- ทำงานร่วมกับงานระบบได้ครบวงจร  
👉 หน้าแนะนำ: `construction.html`
