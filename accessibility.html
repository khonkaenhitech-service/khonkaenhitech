Accessibility Statement

เว็บไซต์ ขอนแก่นไฮเทค ให้ความสำคัญกับการเข้าถึงข้อมูลของผู้ใช้งานทุกกลุ่ม รวมถึงผู้พิการ ผู้สูงอายุ และผู้ใช้อุปกรณ์ช่วยเหลือ

มาตรฐานที่รองรับ

  • WCAG 2.1 ระดับ AAA
  • Google Lighthouse Accessibility
  • รองรับ Screen Reader
  • รองรับ Keyboard Navigation

ฟีเจอร์เพื่อการเข้าถึง

  • คอนทราสต์สีสูง อ่านง่าย
  • Dark Mode (ผู้ใช้เลือกได้)
  • โครงสร้าง HTML ตาม Semantic
  • ข้อความ Alt สำหรับรูปภาพ
